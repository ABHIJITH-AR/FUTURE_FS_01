## ⚙️ Development Guide

### Prerequisites
- Node.js (v18+)
- npm

### Installation & Run
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build the application for production:
   ```bash
   npm run build
   ```
